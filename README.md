  A iOS/macOS client for HackerNews made in SwiftUI

  [![Linux Build][travis-image]][travis-url]

## Features

  * Robust routing
  * Focus on high performance
  * HTTP helpers (redirection, caching, etc)
  * View system supporting 2 template engines
